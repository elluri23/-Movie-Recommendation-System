# -Movie-Recommendation-System
Built a content-based recommendation system using Python and Scikit-learn that suggests similar items based on cosine similarity. This project helped me understand similarity metrics, vectorization techniques, and real-world ML deployment using Streamlit.
# 🎬 Movie Recommendation System

## 📌 Overview
This project builds a content-based recommendation system using cosine similarity.

## 🚀 Features
- Recommends similar movies
- Built with Scikit-learn
- Streamlit Web App

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## ⚙️ How It Works
1. Text vectorization using CountVectorizer
2. Cosine similarity calculation
3. Similar movie ranking

## 🧠 Key Learnings
- Similarity metrics
- Handling missing data

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py
